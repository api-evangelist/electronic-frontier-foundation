# Electronic Frontier FoundationThis is a repo for managing machine readable artifacts for the [Electronic Frontier Foundation APIs](https://www.eff.org/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Electronic Frontier Foundation APIs](https://www.eff.org/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.eff.org/)- [Developer](https://www.eff.org/)- [Twitter](https://twitter.com/eff)- [GitHub](https://github.com/EFForg)The [APIs.json](https://github.com/api-evangelist/electronic-frontier-foundation/blob/master/apis.json) index provides links to all the relevant links for the Electronic Frontier Foundation APIs, including the OpenAPI and Postman if they are present.
